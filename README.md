# 🩺 AI Medical Diagnoser
An AI-powered web application that predicts diseases and suggests prescriptions based on user-provided symptoms using Deep Learning (LSTM + NLP).

## 🚀 Features
* 🔍 Symptom-based disease prediction
* 💊 Prescription recommendation
* 🧠 Deep Learning model (LSTM)
* 🌐 Interactive web app using Streamlit
* ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack
* Python
* TensorFlow / Keras
* NLP (Tokenizer, Padding)
* Streamlit
* Scikit-learn
